Python for Data Analysis

An in-person introductory training program focused on practical data analysis using Python.

Program Overview

This short training program is designed to introduce participants to Python as a tool for data analysis. The course emphasizes hands-on learning and practical understanding of how data can be explored, manipulated, and visualized using Python.

Sessions are conducted in person and are structured to support learners with limited or no prior experience in data analysis.

Program Structure

  • Mode: In-person training
  • Duration: Two (2) weeks
  • Sessions: Two sessions per week
  • Total Sessions: Four (4)
  • Development Environment: Visual Studio Code

Learning Outcomes

By the end of this program, participants will be able to:

  • Understand Python fundamentals relevant to data analysis
  • Work with numerical data using NumPy
  • Load, clean, and explore datasets using Pandas
  • Create basic data visualizations using Matplotlib
  • Use VS Code effectively for Python data projects
